# Monte Carlo 옵션 계산기 (웹)

간단한 Flask 기반 웹 애플리케이션으로 유럽형 콜/풋 옵션 가격을 몬테카를로 시뮬레이션으로 계산합니다. 첫 페이지에서 입력 방법이 바로 안내되며, 사용자가 직접 변수를 수정한 뒤 실행할 수 있습니다.

## 실행 방법

1. 의존성 설치
   ```bash
   pip install -r requirements.txt
   ```
2. 서버 실행
   ```bash
   python app.py
   ```
3. 브라우저에서 <http://localhost:8000> 접속 후 입력값을 넣어 실행합니다.

## 시뮬레이션 모델
- 기하 브라운 운동(GBM) 경로를 생성해 만기 시점의 페이오프를 계산합니다.
- 각 페이오프를 무위험이자율로 할인한 뒤 평균하여 옵션 프리미엄을 추정합니다.
- 시뮬레이션 횟수와 타임스텝을 늘리면 정확도는 올라가지만 계산 시간이 늘어납니다.
