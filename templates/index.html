Monte Carlo Web 옵션 계산기

사용자 입력에 따라 유럽형 콜/풋 옵션 가격을 몬테카를로 시뮬레이션으로 추정합니다.

사용 방법
  1. 아래 폼에 기초자산 가격, 행사가, 만기(년), 무위험이자율, 변동성, 시뮬레이션 횟수, 타임스텝을 입력합니다.
  2. 옵션 종류(Call/Put)를 선택한 뒤 시뮬레이션 실행을 누릅니다.
  3. 결과 카드에서 할인된 옵션 가격을 바로 확인할 수 있습니다.
  4. 값을 바꾸어 여러 시나리오를 테스트하면서 감도를 살펴보세요.

모델 메모
  • 기하 브라운 운동(GBM)을 사용해 가격 경로를 생성합니다.
  • 각 경로의 만기 시점 페이오프를 할인하여 평균을 내 옵션 가치를 추정합니다.
