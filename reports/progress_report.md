# 2025년 10월 프로젝트 중간 보고서

**프로젝트명:** 글로벌 반도체·빅테크 주가 추세 분석 및 예측 시스템 구축

**작성일:** 2025-10-31

**작성자:** 데이터 분석 팀

---

## 1. 경영 요약 (Executive Summary)
- 2023-10-16부터 2025-10-10까지 516영업일 동안 삼성전자(005930.KS), 애플(AAPL), 엔비디아(NVDA)의 일별 시가·고가·저가·종가·거래량을 통합한 원천 데이터를 수집하고 단일 CSV로 정리하였다.
- 데이터 품질 점검 결과, 삼성전자 지표는 총 516건 중 34건(6.59%), 애플과 엔비디아 지표는 각 17건(3.29%)의 결측이 존재함을 확인하고, 결측 패턴을 기반으로 향후 보간 전략을 수립하였다.
- 일별 로그수익률 분석에서 엔비디아 평균 일간 수익률이 0.32%로 가장 높고 변동성(표준편차 3.17%) 역시 가장 높아 리스크-보상 프로파일이 뚜렷함을 파악하였다. 반면 삼성전자는 평균 0.05%, 변동성 1.91%로 상대적으로 안정적이다.
- AAPL과 NVDA 로그수익률 간 상관계수는 0.363으로, 동일 산업군 내 동조화를 시사한다. 삼성전자와의 상관은 0.024~ -0.027로 낮아, 글로벌 분산투자 효과가 기대된다.
- 데이터 진단, 품질 관리, 기본 통계량 및 수익률 기반 EDA(Exploratory Data Analysis)를 완료했고, 예측 모델링을 위한 특징 엔지니어링 로드맵을 수립했다.

## 2. 프로젝트 목표 및 범위
1. **단기 목표:** 고품질의 시계열 데이터셋을 확보하고, 가격 및 거래량 특성을 정량화하여 예측 모델 설계에 필요한 통찰을 도출한다.
2. **중기 목표:** 주가 예측을 위한 다중 입력(가격·거래량·파생지표) 특징 세트를 구축하고, 베이스라인 모델(ARIMA, Prophet, LSTM 등)을 비교 평가한다.
3. **장기 목표:** 투자 시뮬레이션 모듈을 포함하는 의사결정 지원 시스템을 구현하고, 실시간 데이터 스트리밍 파이프라인으로 확장한다.

## 3. 수행 작업 상세

### 3.1 데이터 수집 및 적재
- 삼성전자, 애플, 엔비디아의 일별 시세를 동일한 스키마로 수집한 뒤, `temp.csv` 파일로 통합하였다. 컬럼은 날짜와 각 종목별 종가·고가·저가·시가·거래량으로 구성한다.
- 파일 구조를 점검하여 멀티 헤더(가격/종목 구분)가 포함된 것을 확인하고, 파이썬 CSV 리더로 커스텀 파서 로직을 작성해 분석용 딕셔너리 형태로 적재하였다.
- 수집 범위는 2023년 10월 16일부터 2025년 10월 10일까지 총 516개의 거래일을 포함한다.

### 3.2 데이터 품질 점검 및 결측 분석
- 각 종목·지표별 전체 카운트를 확인하고, 결측(빈 문자열) 발생 횟수와 비율을 산출하였다.
- 결측치는 특정 날짜 구간에 집중되어 있음을 확인했으며, 거래량과 가격 지표가 동시에 비어 있는 형태가 대부분이라 연속 데이터 보간(forward/backward fill) 및 외부 데이터 대체 방안을 검토 중이다.
- 데이터 타입은 모두 실수(float)로 정상 변환되었으며, 음수 값이나 0 이하 값은 발견되지 않았다.

### 3.3 기술 통계량 및 변동성 프로파일링
- 각 종목별 종가·시가·고가·저가·거래량의 평균, 표준편차, 최소·최대값을 산출하여 가격 수준과 변동성 범위를 파악하였다.
- 엔비디아는 평균 종가 115.60달러, 표준편차 38.97달러로 성장주 특성이, 애플은 평균 종가 209.52달러, 표준편차 24.35달러로 안정적 대형주 특성이 확인되었다. 삼성전자는 평균 종가 66,472원, 표준편차 9,699원으로 코스피 대형주의 변동성이 반영되었다.
- 거래량 분석에서는 엔비디아 평균 3.25억 주, 애플 평균 5.66천만 주, 삼성전자 평균 1,948만 주로 시장 규모 차이를 계량화하였다.

### 3.4 수익률 시계열 분석
- 전일 대비 로그수익률을 계산하여 수익률 분포, 평균 수익률, 변동성(표준편차)을 측정하였다.
- 엔비디아의 평균 로그수익률은 0.00320(0.32%), 삼성전자는 0.00051(0.05%), 애플은 0.00090(0.09%)이며, 변동성은 각각 0.0317, 0.0191, 0.0175로 측정되었다.
- 로그수익률 기반 상관분석을 수행해 AAPL-NVDA는 0.363으로 강한 양의 상관관계를, AAPL-삼성전자는 0.024, NVDA-삼성전자는 -0.027로 약한 상관 또는 약한 음의 상관을 확인했다.

### 3.5 리포팅 및 문서화
- 위 분석 결과를 정리한 중간 보고서를 작성하여 이해관계자 공유를 준비했다.
- 데이터 파이프라인과 분석 결과를 추적할 수 있도록, 수집·전처리·탐색·분석 단계를 문서화하였다.

## 5. 기술적 인프라 및 작업 환경

- 분석 코드는 버전 관리 시스템(Git)을 통해 추적하고 있으며, 데이터 파일(`temp.csv`)과 분석 산출물(`reports/progress_report.md`)을 함께 관리하여 재현성과 협업 효율을 확보하였다.
- 향후 확장성을 고려해 Pandas, NumPy 등 과학 연산 라이브러리를 도입할 수 있도록 모듈화된 코드 구조를 설계 중이다.

## 6. 주요 이슈 및 대응 전략
1. **결측치 처리 전략 필요:** 동일 날짜에 여러 지표가 동시에 결측되는 케이스가 존재한다. *대응:* 최근접 유효 값 보간과 외부 데이터 소스(예: 거래소 공시 데이터) 대체를 병행하는 하이브리드 전략을 설계한다.
2. **변동성 이질성:** 엔비디아의 고변동성으로 인해 공통 모델 적용 시 과적합 또는 예측 오차 확대 가능성이 있다. *대응:* 변동성 조정(feature scaling) 및 개별 종목 모델링을 병행 검토한다.
3. **스케일 차이:** 삼성전자(원화)와 애플·엔비디아(달러) 간 단위 차이가 크다. *대응:* 환율 보정을 통해 동일 통화 기준으로 재표현하거나, 수익률 기반 비교 지표로 통일한다.

## 7. 향후 계획 및 로드맵
- **단기 (1~2주):** 결측치 보간 로직 구현, 이상치 탐지(박스플롯/IQR, Z-Score) 수행, 파생지표(이동평균, RSI, 거래량 변화율 등) 생성.
- **중기 (3~6주):** ARIMA/Prophet/LSTM을 활용한 다중 예측 모델 프로토타입 구현 및 백테스트 프레임워크 구성.
- **장기 (6주 이후):** 실시간 데이터 수집 파이프라인 구축, 리스크 관리 지표(최대 낙폭, 샤프 지수 등) 통합, 대시보드 시각화 배포.

### 부록 A. 데이터 처리 파이프라인 개요
1. **로드 단계:** CSV 파일의 다중 헤더 구조를 파악하고, 날짜/종목/지표별 키를 생성해 메모리 내 딕셔너리로 변환.
2. **검증 단계:** 날짜 파싱 실패 레코드를 필터링하고, 각 지표별 결측 횟수·비율을 계산.
3. **EDA 단계:** 기술 통계량, 로그수익률, 상관계수를 산출해 특징 탐색.
4. **문서화 단계:** 분석 결과와 향후 작업 계획을 정리한 보고서를 작성하여 지식 베이스에 추가.
